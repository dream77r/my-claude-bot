# Обещания команды

Трекер того, кто что обещал сделать. Обновляется автоматически при утренних
стендапах и по ходу чата, когда участники берут на себя задачи.

## Формат записи

Каждое обещание — отдельный пункт в разделе по дате. Используй формат:

```
## YYYY-MM-DD

- [ ] **@username** — что обещал сделать — дедлайн: YYYY-MM-DD HH:MM — добавлено: ISO timestamp
      └ источник: ссылка на сообщение или контекст
      └ статус: обновляется вечером
```

Когда задача выполнена, замени `[ ]` на `[x]` и добавь строку:
`      └ закрыто: ISO timestamp — комментарий`.

Если задача не выполнена вечером и переносится, добавь строку:
`      └ перенесено на YYYY-MM-DD — причина`.
