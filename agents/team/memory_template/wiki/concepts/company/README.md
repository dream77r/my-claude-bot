# База знаний компании

Это папка для хранения общей информации о компании, продукте и команде,
которая используется ботом для ответов на вопросы участников чата.

## Как наполнять

Кладите сюда markdown-файлы с фактами, которые бот должен знать.
Структура — произвольная, но рекомендуется:

- `mission.md` — миссия, продукт, основные метрики
- `team.md` — кто в команде, роли, зоны ответственности
- `processes.md` — рабочие процессы, ритуалы, договорённости
- `glossary.md` — термины и сокращения
- `links.md` — важные ссылки (доки, дашборды, репозитории)
- `faq.md` — часто задаваемые вопросы и ответы на них

## Откуда бот это читает

Бот подгружает файлы из этой папки в контекст, когда участник чата задаёт
вопрос, триггерящий скилл `knowledge-base`. Также содержимое индексируется
в `../index.md` при регулярном Dream-проходе.

## Приватность

Всё, что лежит здесь, доступно всем участникам чата, где работает бот.
Не кладите сюда секреты, пароли, токены.
