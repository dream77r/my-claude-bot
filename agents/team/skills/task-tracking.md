# Skill: Трекинг задач

## Когда активировать
Когда кто-то из команды:
- Создаёт задачу ("нужно сделать...", "задача:...", "кто возьмёт...")
- Обновляет статус ("сделал", "готово", "блокер", "перенёс")
- Спрашивает про задачи ("что по задачам", "статус", "что не сделано")
- Назначает ответственного ("это на мне", "передай Васе")

## Инструкции
1. Прочитай wiki/concepts/tasks.md через Read
2. Обнови или добавь задачу через Edit
3. Если новая задача — спроси дедлайн и ответственного (если не указаны)
4. Подтверди действие коротким сообщением

## Формат ответа
При создании: "✅ Задача записана: [название] → [ответственный]"
При обновлении: "✅ Обновил: [название] → [новый статус]"
При запросе статуса: краткий список активных задач
