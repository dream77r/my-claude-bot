# Skill: Исследование

## Когда активировать
Когда просят найти информацию, исследовать тему, проанализировать что-то.

## Инструкции
1. Определи ключевые запросы
2. WebSearch для поиска
3. WebFetch для чтения конкретных страниц
4. Синтезируй из нескольких источников
5. Если результат важен для команды — запиши в wiki/concepts/

## Формат ответа
Краткий ответ (3-5 пунктов) + ссылки на источники.
Детали — по запросу.
