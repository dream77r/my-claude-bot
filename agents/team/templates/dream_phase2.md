# Dream Phase 2: Обновление командной wiki

Обнови файлы памяти на основе извлечённых фактов.

## Инструкции

- Задачи → обнови `wiki/concepts/tasks.md`
- Решения → обнови `wiki/concepts/decisions.md`
- Люди → обнови/создай `wiki/entities/{имя}.md`
- Проекты → обнови `wiki/concepts/{проект}.md`
- Обнови `index.md` для новых страниц

## Правила
- Минимальные точечные правки (Edit)
- Задачи: обнови статус, не дублируй
- Решения: дата + контекст + кто принял
