# Evening Digest Template

Вечерний дайджест: подведение итогов дня для группы. Сравнение план/факт и подготовка переносов на завтра.

## Что прочитать перед началом

1. `groups/{chat_id}/settings.json` — `message_thread_id`, `language`.
2. `groups/{chat_id}/daily/YYYY-MM-DD.md` — сырой лог всех сообщений за сегодня.
3. `wiki/concepts/promises.md` — сегодняшняя секция.
4. `wiki/concepts/tasks.md` — обновления задач за сегодня (по `git log -p`).

## Шаги обработки

1. **Определи выполненные обещания.**
   Для каждой `[ ]` в сегодняшней секции `promises.md`:
   - Просканируй `daily/YYYY-MM-DD.md` на паттерны «сделал X», «закрыл X», «готово», «done», где X семантически совпадает с обещанием.
   - Если совпадает — пометь `[x]` и добавь строку `└ закрыто: ISO — <цитата из чата>`.
   - Если не совпадает и дедлайн сегодня — оставь `[ ]`, добавь в список «не закрыто».

2. **Перенеси невыполненные на завтра.**
   Для каждого оставшегося `[ ]` с дедлайном сегодня:
   - Добавь строку `└ перенесено на YYYY-MM-DD — причина: не отмечено как выполненное`.
   - Создай (если нет) секцию `## YYYY-MM-DD` на завтра в `promises.md` и скопируй туда запись с новым `добавлено:` timestamp.

3. **Собери статистику дня.**
   - `закрыто: N`
   - `перенесено: M`
   - `новые задачи: K` (из `tasks.md` за сегодня)
   - `принятые решения: L` (из `decisions.md` за сегодня)

## Шаблон сообщения

### Русский

```
🌙 **Итоги дня**

✅ Закрыто: {N}
{список: - @user — задача}

⏳ Переносим на завтра: {M}
{список: - @user — задача}

{если решений > 0: "📝 Решения дня: {L}"}
{если новых задач > 0: "🆕 Новых задач: {K}"}

До завтра! ☀️
```

### English

```
🌙 **End-of-day digest**

✅ Closed: {N}
{list: - @user — task}

⏳ Rolling to tomorrow: {M}
{list: - @user — task}

{if decisions > 0: "📝 Decisions today: {L}"}
{if new tasks > 0: "🆕 New tasks: {K}"}

See you tomorrow! ☀️
```

## Действия после формирования

1. Запиши `dispatch/{chat_id}_{thread_id}_{iso}.json` с `source: "evening_digest"`. Dispatcher заберёт файл на следующем поллинге и опубликует в Telegram.

2. Обнови `wiki/concepts/promises.md` через Edit (отметки закрытия и переносы).

3. Обнови `wiki/synthesis/daily/YYYY-MM-DD.md`:
   ```markdown
   ## Вечерний дайджест
   - время: ISO
   - закрыто: N
   - перенесено: M
   - новых задач: K
   - решений: L
   - ссылки на страницы обновлённых задач
   ```

4. Добавь в `log.md`.

## Ограничения

- Если сегодня стендапа не было — дайджест всё равно запускай, просто без сравнения план/факт (только статистика по чату).
- Если `daily/YYYY-MM-DD.md` пустой — отправь короткое «Сегодня тихий день, ничего не обсуждали».
- Не отправляй пустой дайджест с нулями во всех метриках — заменяй на короткое «Сегодня без движения».
- Не дублируй (проверь секцию в `wiki/synthesis/daily/YYYY-MM-DD.md`).
