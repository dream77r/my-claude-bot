# Примеры доменов

> Готовые доменные схемы, которые можно взять за основу при инициализации
> нового vault'а. Каждый пример — **полностью анонимизированный**, без реальных
> имён, ИНН, адресов и цифр.

Когда пользователь запускает `vault-init` в режиме C (от примера), Архивариус
предлагает выбрать один из этих шаблонов и копирует его структуру в
`memory/` пользователя.

## Доступные примеры

- **small-business/** — малый бизнес с несколькими юрлицами, точками,
  контрагентами и договорами. Подходит для розницы, общепита, услуг, любой
  деятельности с несколькими ООО и сетью контрагентов.

_Добавляйте новые примеры пулл-реквестами. Требования: все имена-плейсхолдеры
(`ACME Corp`, `Supplier A`), ИНН из нулей или `0000000000`, адреса —
`Example Street, 1`, суммы — `0.00`._
