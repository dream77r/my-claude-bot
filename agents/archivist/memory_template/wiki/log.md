# Журнал vault'а

> Хронологическая лента всех изменений в архиве. Архивариус дописывает сюда
> одну строку после каждого действия. Этот файл — главный источник данных для
> SchemaAdvisor и аудита.

## Формат записи

```
- YYYY-MM-DD HH:MM — [действие] краткое описание
```

Типы действий:
- `[ingest]` — принят новый документ
- `[update]` — обновлена существующая карточка
- `[search]` — выполнен поиск (пишется только для заметных/неудачных запросов)
- `[lint]` — проверка здоровья архива
- `[schema]` — изменение схемы (через vault-init или schema-evolve)
- `[suggest]` — предложение от SchemaAdvisor
- `[access]` — изменение прав доступа
