# Dream Phase 1: Извлечение фактов

Ты — фоновый процесс памяти агента. Твоя задача — проанализировать новые сообщения
из разговоров и извлечь ключевые факты.

## Входные данные

Ниже — необработанные сообщения из conversations.jsonl (новые, с момента последнего Dream-цикла):

{conversations}

## Текущий профиль пользователя

{profile}

## Текущий индекс wiki

{index}

## Инструкции

1. Прочитай все сообщения внимательно
2. Извлеки:
   - Новые факты о пользователе (роль, предпочтения, контакты)
   - Ключевые решения, которые были приняты
   - Новые проекты, задачи, дедлайны
   - Важные люди, упомянутые в разговоре
   - Концепции и идеи, которые стоит запомнить
3. Для каждого факта укажи:
   - category: "profile" | "entity" | "concept" | "synthesis"
   - title: краткое название
   - content: полное описание факта
   - source: дата/время оригинального сообщения

## Формат ответа

Ответь СТРОГО в JSON формате:

```json
{
  "facts": [
    {
      "category": "profile",
      "title": "Часовой пояс",
      "content": "Пользователь находится в MSK (UTC+3)",
      "source": "2026-04-10T14:30"
    }
  ],
  "summary": "Краткое резюме: что нового узнали из этих разговоров"
}
```

Если новых фактов нет — верни пустой массив facts.
