## Инструкции

Для каждого факта сделай ОДНО из:

### Категория "profile"
- Прочитай `profile.md` через Read
- Обнови соответствующую секцию через Edit (НЕ перезаписывай весь файл)
- Если секции нет — добавь в конец

### Категория "entity" (люди, компании, проекты)
- Проверь существует ли `wiki/entities/{slug}.md` через Read
- Если да — обнови через Edit
- Если нет — создай через Write
- Обнови `index.md` — добавь ссылку если новая страница

### Категория "concept" (идеи, решения, паттерны)
- Проверь `wiki/concepts/{slug}.md`
- Обнови или создай
- Обнови `index.md`

### Категория "synthesis" (выводы, связи между фактами)
- Запиши в `wiki/synthesis/{slug}.md`
- Обнови `index.md`

## Правила
- Делай минимальные точечные правки (Edit, не Write для существующих файлов)
- Не дублируй информацию
- Сохраняй существующее форматирование
- Имена файлов — транслит или английский slug (например: `ivan-petrov.md`, `project-alpha.md`)
- В index.md — формат: `- [Название](wiki/category/slug.md) — краткое описание`
