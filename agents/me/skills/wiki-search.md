# Skill: wiki-search

## Когда активировать

- Команда `/recall <тема>` — всегда триггерит этот навык.
- Любой вопрос вида «что мы обсуждали про X», «когда мы говорили о Y»,
  «вспомни про Z», «найди в памяти ...».

## Зачем

Мастер-агент не должен галлюцинировать про прошлые разговоры. У нас есть
`./wiki/` (markdown-страницы), `./graph.json` (граф связей) и `./daily/`
(дневные логи). Этот навык запускает детерминированный поиск по ним —
BM25 + BFS — и подаёт тебе уже отфильтрованный контекст. Ты только
форматируешь человеческий ответ.

## Алгоритм

1. Извлеки тему из запроса. Например:
   - `/recall Phase 5` → `Phase 5`
   - `что мы обсуждали про MCP` → `MCP`

2. Запусти Bash-команду из корня проекта:
   ```bash
   cd ~/my-claude-bot && python3 -m src.wiki_search \
     --agent agents/me \
     --query "<тема>" \
     --top 5
   ```

3. Команда возвращает JSON вида:
   ```json
   {
     "query": "...",
     "hits": [
       {
         "name": "Phase5",
         "path": "wiki/concepts/Phase5.md",
         "type": "concepts",
         "score": 1.23,
         "snippet": "...кусок страницы вокруг совпадения...",
         "neighbors": ["Иван", "MCP"],
         "quote": "...цитата из daily...",
         "quote_date": "2026-04-12"
       }
     ],
     "extra_neighbors": ["..."]
   }
   ```

4. Сформулируй ответ пользователю на русском по такому шаблону:

   ```
   🧠 По теме «<query>» нашёл в памяти:

   • **<name>** (<type>, последний раз — <quote_date>)
     <короткий пересказ snippet своими словами>
     Связано с: <neighbors через запятую>
     Цитата: «<quote>»
   ```

5. Если `hits` пустой — честно скажи «В памяти пока нет упоминаний <query>».
   Не выдумывай. Не дополняй из своих знаний — это команда recall, она
   должна работать только с реальной памятью.

6. Если `extra_neighbors` непустой — добавь в конце строку
   «Также рядом в графе: …».

## Что НЕ делать

- Не вызывай этот навык на каждый вопрос — только когда пользователь
  явно просит вспомнить, или ввёл `/recall`.
- Не редактируй wiki/ из этого навыка. Recall — read-only.
- Не запускай BM25 руками через Read/Grep — используй CLI, он уже
  оптимизирован и возвращает ранжированный результат.
- Не галлюцинируй: если в `quote` пусто — так и пиши, цитат из daily нет.
