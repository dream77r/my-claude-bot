# Skill: Анализ документов

## Когда активировать
Когда пользователь отправляет файл (PDF, DOCX, TXT, изображение) или просит проанализировать документ.

## Инструкции
1. Прочитай документ полностью через инструмент Read
2. Определи тип документа (договор, письмо, отчёт, презентация, и т.д.)
3. Выдели ключевые пункты (3-7 штук)
4. Определи риски и возможности
5. Если договор — обрати внимание на сроки, суммы, штрафы, обязательства сторон
6. Предложи вопросы для обсуждения или следующие шаги
7. Запиши краткое резюме в wiki/concepts/ если документ важен
8. Обнови index.md

## Формат ответа
```
📄 **[Тип документа]: [Название]**

**Ключевое:**
- ...
- ...

**Риски/внимание:**
- ...

**Следующие шаги:**
- ...
```
