# Skill: Веб-исследование

## Когда активировать
Когда пользователь просит найти информацию, исследовать тему, проверить факт, или когда нужны актуальные данные.

## Инструкции
1. Определи ключевые поисковые запросы
2. Используй WebSearch для поиска
3. Используй WebFetch для чтения конкретных страниц
4. Синтезируй информацию из нескольких источников
5. Укажи источники (ссылки)
6. Если информация важна — запиши в wiki/concepts/

## Формат ответа
```
🔍 **[Тема исследования]**

**Результат:**
- ...

**Источники:**
- [название](url)

**Вывод:**
...
```

## Ограничения
- Всегда указывай источники
- Различай факты и мнения
- Если данные устарели — предупреди
- Не выдавай предположения за факты
